# ross

RSS/Atom fetcher.  It takes a list of feed URLs as input, and outputs a list of
articles from the last week.  I email it to myself once a week, so I don't get
disturbed by checking RSS app every day.

Work in progress.

## Usage

lein run <FEEDS-FILE>
